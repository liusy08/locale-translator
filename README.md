# 🌐 本地化文件翻译工具

一个功能强大的本地化文件（.cfg格式）翻译管理工具，支持批量翻译、智能检测和一键导出。

## ✨ 功能特性

### 📥 导入功能
- **导入本地化文件**：支持导入 `.cfg` 格式的本地化文件
- **导入已翻译文件**：可选，导入已有的翻译文件继续编辑

### 🔄 翻译功能
- **单条翻译**：点击任意条目的「翻译」按钮，自动调用 MyMemory 免费 API 翻译并填入
- **批量翻译**：点击区块标题旁的「🔄 批量翻译」按钮，自动翻译该区块所有未翻译的条目

### 🔍 智能检测
- **相同键建议**：当焦点在翻译输入框时，自动显示其他区块相同键的翻译建议
- **不一致筛选**：检测同一键在不同区块中翻译不一致的条目
- **包含关系检测**：检测原文包含关系与翻译包含关系不匹配的问题

### 🎨 界面功能
- **区块切换**：Tab 标签快速切换不同区块
- **筛选功能**：支持「全部」「未翻译」「已翻译」「不一致」「包含关系」五种筛选模式
- **搜索功能**：支持按键名、原值、翻译值搜索
- **进度统计**：实时显示翻译进度

### 📤 导出功能
- **一键导出**：导出翻译后的本地化文件为 `.cfg` 格式

## 🚀 使用方法

### 1. 导入文件
点击「导入本地化文件」按钮，选择要翻译的 `.cfg` 文件。

### 2. 开始翻译
- **单条翻译**：点击条目旁的「翻译」按钮
- **批量翻译**：点击区块标题旁的「🔄 批量翻译」按钮

### 3. 手动调整
翻译完成后可手动修改不满意的结果。

### 4. 导出结果
点击「💾 导出翻译文件」保存翻译结果。

## 🔧 技术说明

### 翻译 API
使用 **MyMemory** 免费翻译 API：
- 无需 API 密钥
- 从英文(English)翻译为简体中文(Chinese Simplified)
- 每日有字数和频率限制

### 文件格式
支持 Factorio 风格的 `.cfg` 本地化文件格式：
```cfg
[section-name]
key=value
another-key=another-value
```

## 💡 快捷操作

| 操作 | 说明 |
|------|------|
| 点击「翻译」 | 自动翻译当前条目 |
| 点击「批量翻译」 | 自动翻译当前区块所有未翻译条目 |
| 点击「复制」 | 复制原文到剪贴板 |
| 输入框获得焦点 | 显示相同键的其他翻译建议 |
| 点击筛选标签 | 切换筛选模式 |

## ⚠️ 注意事项

1. **翻译限制**：MyMemory 免费版有每日字数和请求频率限制，大批量翻译建议分批进行
2. **翻译质量**：建议翻译完成后人工检查，确保翻译质量
3. **数据备份**：重要翻译任务请定期导出备份
4. **浏览器要求**：需要支持 JavaScript 和 Fetch API
